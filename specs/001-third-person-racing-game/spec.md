# 功能規格：第三人稱賽車遊戲

**分支**: `001-third-person-racing-game`  
**日期**: 2026-03-13  
**狀態**: 草稿

---

## 需求摘要

建立一款可在瀏覽器執行、可部署至 GitHub Pages 的第三人稱賽車遊戲。玩家從車輛後方視角（第三人稱）駕駛賽車在封閉賽道上行駛，完成計圈，並追求最快單圈時間。

---

## 功能需求

### 核心功能
1. **第三人稱相機**：相機跟隨賽車，位於車輛後上方，可平滑追蹤轉向
2. **車輛控制**：鍵盤方向鍵或 WASD 控制加速、煞車、左右轉向
3. **賽道**：封閉環形賽道，具有邊界碰撞檢測
4. **計圈系統**：記錄當前圈數與最佳單圈時間
5. **HUD（平視顯示器）**：顯示速度、圈數、計時器、最佳圈速

### 次要功能
6. **出發倒數**：3-2-1-GO! 倒數動畫
7. **完賽畫面**：顯示最終成績（總時間、最佳圈速）

---

## 非功能需求

- 目標幀率：60 FPS（現代桌面瀏覽器）
- 無後端服務，純靜態前端
- 可部署至 GitHub Pages（`/docs` 目錄或 `gh-pages` 分支）
- 支援主流桌面瀏覽器（Chrome、Firefox、Edge）

---

## 範圍外

- 多人連線對戰
- 行動裝置觸控控制
- AI 對手賽車
- 音效系統（MVP 階段不含）
- 儲存排行榜至後端

---

## 驗收標準

1. 頁面載入後顯示 3D 賽車場景，相機位於賽車後上方
2. 按下方向鍵賽車可前後左右移動，相機跟隨
3. 賽車無法穿越賽道邊界
4. HUD 顯示即時速度與計時
5. 完成一圈後圈數 +1，並記錄單圈時間
6. 部署後可透過 GitHub Pages URL 正常開啟遊戲
